About

Hey, I’m Sam. I’m a college student pursuing a BS in computer science at Arkansas State University in Jonesboro, Arkansas. I’m a senior, expecting to graduate in the Spring of 2019. I have been studying web development and computer science since high school. I enjoy the experience of creating a project from scratch and then witnessing it improve someone’s life.

Since May of 2018, I have been a part of a web development internship with AceOne Technologies. I’ve been challenged with an array of projects utilizing frontend and backend technologies. I’ve been apart of projects for developing a web application to track an employee’s time spent working, a website to purchase online courses, and a REST API for a mobile application. This experience has been invaluable, especially since I’m still in college.

Projects

Life Organizer

I created this project for my Database Systems class in the Fall of 2018. I designed a web application to organize a user's life. They can track their daily tasks, habits, goals and notes. When user's complete tasks, goals, or habits they earn experience points that enable them to level up.

I completed this project using Bootstrap for the front-end interface, PHP for the back-end programming, and PL/SQL for the database.

Technologies: PHP, PL/SQL, Bootstrap

Clothes Pen

This project was created for a web design competition in PBL (Phi Beta Lambda). My partner and I were tasked to design a website for an imaginary, non-profit clothing store. We implemented the website with a playful color scheme, minimalistic design, interactive interface, responsive capabilities, and informative content.

My duties were to help visualize the design, create the mockups, and construct the website using HTML, CSS, and Javascript.

This website won first at the Arkansas State PBL Conference and then went on to win second place at the 2018 PBL National Leadership Conference in Baltimore, MD.

Technologies: HTML5, CSS3, SASS, JS
